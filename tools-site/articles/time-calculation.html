時間計算の基本と応用：勤務時間・残業代・時差まで完全ガイド

📅 2024年1月 ⏱️ 読了時間：約8分 🏷️ 時間・勤務・残業・時差

時間の計算は算数の中でも少し特殊です。「10時30分から18時45分まで何時間何分か」「時給1,200円で8時間15分働いたらいくらもらえるか」──こういった計算は普通の足し算引き算と少し勝手が違います。この記事では、時間計算の基本から実用的な応用まで丁寧に解説します。

時間計算の基本：60進法を理解する

普段の算数は10進法（0〜9の10種類の数字を使い、10になったら桁が上がる）ですが、時間は60進法（60秒＝1分、60分＝1時間）です。そのため、普通の計算とは違うルールが必要です。

時間の足し算

3時間45分 + 2時間30分を計算してみましょう。

  • 分の計算：45 + 30 = 75分 → 75分 = 1時間15分（60分を1時間に繰り上げ）
  • 時間の計算：3 + 2 + 1（繰り上げ）= 6時間
  • 答え：6時間15分

時間の引き算

5時間20分 − 2時間50分を計算します。

  • 分の計算：20 − 50ができないので、1時間借りて 80 − 50 = 30分
  • 時間の計算：(5−1) − 2 = 2時間
  • 答え：2時間30分

時刻から経過時間を求める

10時30分から18時45分まで何時間何分か？

  • 方法1：18時45分 − 10時30分 = 8時間15分（直接引く）
  • 方法2：10時30分 → 18時30分 = 8時間、18時30分 → 18時45分 = 15分 → 合計8時間15分

日をまたぐ時間計算

夜勤・徹夜作業など、日をまたいで計算する場合は少し工夫が必要です。

📐 計算方法：前日の終了時刻が翌日の場合、終了時刻に24時間を加えて計算します。
例：22時00分 ～ 翌4時30分 → 28時30分（24+4:30）− 22時00分 = 6時間30分

勤務時間と賃金の計算

時給から日給・月給を計算する

時給1,200円で1日8時間、月20日働いた場合の給与：

  • 日給 = 1,200円 × 8時間 = 9,600円
  • 月給 = 9,600円 × 20日 = 192,000円

端数（分）が出る場合の計算

時給1,200円で8時間15分働いた場合：

  • 8時間15分 = 8.25時間（15分 ÷ 60 = 0.25）
  • 給与 = 1,200円 × 8.25時間 = 9,900円

残業代の計算方法

日本の労働基準法では、1日8時間・週40時間を超えた労働に対しては割増賃金の支払いが義務付けられています。

残業の種類割増率根拠
時間外労働（法定時間外）25%以上増労基法37条
深夜労働（22時〜5時）25%以上増通常の深夜割増
時間外＋深夜50%以上増両方適用
休日労働（法定休日）35%以上増法定休日の場合
月60時間超の時間外（大企業）50%以上増2023年4月から中小企業も適用

残業代計算の具体例

月給25万円・所定労働時間160時間/月の会社員が20時間残業した場合：

  • 1時間あたりの賃金 = 250,000 ÷ 160 = 1,562.5円
  • 残業割増単価 = 1,562.5 × 1.25 = 1,953円（25%割増）
  • 残業代 = 1,953円 × 20時間 = 39,062円

夏時間（サマータイム）に注意

欧米諸国は春〜秋に夏時間（サマータイム）を採用しているため、時差が季節によって変わります。アメリカでは3月第2日曜〜11月第1日曜、ヨーロッパでは3月最終日曜〜10月最終日曜が夏時間期間です。

日常の時間管理に役立つ計算

通勤時間から逆算する出発時刻

「会社に9時までに到着したい。通勤時間は45分。電車の乗り換え時間が10分かかる場合、何時に家を出ればいい？」

9時00分 − 45分（通勤）− 10分（乗り換え）= 8時05分 → 8時05分より前に出発が必要

作業時間の見積もり

プロジェクト管理や勉強計画では、タスクの作業時間を正確に見積もることが大切です。

  • 過去の類似作業の記録をつける
  • 見積もりに「バッファ（余裕時間）」を20〜30%加える
  • 集中できる時間は個人差があり、一般的に90分が限界とされる
